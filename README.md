# Base44 Labor Management System

This repository contains the backend logic, database schema, forecasting engine, and documentation for the Labor Management System built in Base44.

## Contents

- /docs — architecture, migration guides, UI wireframes
- /database — SQL schema and migration scripts
- /functions — forecasting logic for serverless deployment
- /assets — logos and icons
- /api — API specifications for Base44 integrations

## Purpose

This repo serves as the source of truth for:

- Forecasting logic
- Database structure
- API endpoints
- Documentation
- Version control for Base44 app development

## Deployment

Base44 connects to this repository for:
- Serverless forecasting functions
- SQL migrations
- Assets
- Documentation
